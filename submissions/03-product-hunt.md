## 10. Maker response templates — first 20 likely comments

### "How is this different from [X compliance SaaS]?"

```
Great question. Three differences:
1. We're skills-first, not SaaS-first. You install once into Claude Code, your existing AI agent does the work — no separate dashboard to learn.
2. Open-source MIT. Audit the prompts, fork, modify.
3. Cited answers. Every verdict links back to primary regulation text. No black-box "compliance score."

Happy to do a side-by-side if you have a specific tool in mind.
```

### "Does this give legal advice?"

```
No — and we're explicit about that everywhere. The skills query a regulatory dataset and cite sources. They don't replace a regulatory affairs professional for high-stakes launches. Think of them as the first 90% of the work — the cited evidence you bring to your RA team or attorney.
```

### "What if I don't have a Cleo Legal API key?"

```
The skills work without one. They degrade to public sources (EUR-Lex, eCFR, ECHA, etc.) and tell you clearly when they're in degraded mode. The free API tier (50 calls/day) just lifts the ceiling — you get higher coverage and faster responses.
```

### "How accurate is the regulatory data?"

```
We index 13 primary regulatory databases (EUR-Lex, ECHA, FDA, CPSC, etc.) plus the most-cited industry guidance docs. Updates are monthly for slow-moving sources, daily for fast-moving ones (signals, recalls). Every answer is timestamped and shows source date. If you spot a stale citation, file an issue on the repo — we triage within 24h.
```

### "Is this only for cosmetics?"

```
No — cosmetics is the biggest vertical (8 of 28 skills) but the bundle also covers food, supplements, electronics, toys, textiles, packaging, sustainability. Tier 5 has 8 vertical skills. Tier 1-4 are vertical-agnostic.
```

### "Can I use this on Cursor / Continue / [other AI tool]?"

```
The skills are formatted for Claude Code today. The MCP backend (Cleo Legal API) works with any MCP-compatible client — Cursor, Continue, Claude Desktop, all supported. We're shipping a Cursor-native version Q3. Star the repo to get notified.
```

### "Why €349/mo for Pro? That seems high."

```
Fair question. Our Pro buyer is a brand with €1–10M annual revenue who would otherwise pay a consultant €5–50k per market — €349/mo replaces that with a self-serve workflow. We considered cheaper, but the data infrastructure (13 DBs, daily updates, citations) doesn't break even at €99. We chose to be honest about the price rather than play discount games. Free tier is 50 calls/day forever; that covers most pre-launch needs.
```

### "Self-hosted option?"

```
Not yet. Self-hosting the data pipeline would require shipping the regulatory dataset, which has license restrictions (EUR-Lex is fine, some industry guidance has redistribution rules). We're working on a "bring your own regulator account" mode for enterprise. Drop me a note at contact@cleolabs.co if you're interested.
```

### "Where is Cleo Labs based?"

```
Paris, France. SAS Cleo Corp, SIREN 984567883. We're an EU company building for EU + US + UK + intl markets. Means we know EU regulation cold (CSRD, GPSR, REACH) and treat US/UK as equal partners, not afterthoughts.
```

### "Open-source AND a paid API — what stops you from closing the open-source later?"

```
The MIT license. We can't revoke it on existing versions. New versions could go closed, but the community could fork. We've made a public 24-month maintenance commitment in the repo SECURITY.md. Trust is the moat.
```

### "Why Claude Code specifically?"

```
Two reasons: (1) Claude is the best at long-context regulatory reading (we tested all major models on EUR-Lex passages — Claude wins). (2) Anthropic's skill spec is the cleanest agent-tool primitive shipping today. We'll port to Cursor and OpenAI agents next.
```

### "Show me a real customer."

```
Can't name names on PH (NDA). But here's the unattributed pattern: small EU cosmetics brand (20 SKUs, €800k/yr), used to pay a regulatory consultant €18k/yr. Switched to Cleo Pro at €349/mo. ROI is ~4x year-1, also faster (consultant turnaround was 2 weeks, ours is minutes). Case study going up on the blog post-launch.
```

### "How can I contribute?"

```
Three high-value contributions:
1. Add a skill (template in CONTRIBUTING.md, 1-2h work)
2. Fix a citation that's gone stale (file an issue with the citation URL)
3. Translate the README to your language (we have French + English; need DE, ES, IT, JA)

PRs reviewed within 48h. Naomie is the single approver for now.
```

### "Anthropic plugin registry submission?"

```
Yes — PR is in flight. https://github.com/anthropics/claude-code/pull/[number after the PR opens]. If you upvote that PR's reactions, that genuinely helps the review priority.
```

### "What's the catch?"

```
There isn't one we're hiding. The catch is the same as any open-core model: the free tier has rate limits (50 calls/day). If you stay under, you can use this forever for €0. If you go over, you pay. That's the only "catch."
```

### "Can I see the prompts?"

```
Yes — every skill's prompt is in plain text in the repo at /skills/[skill-id]/SKILL.md. No hidden prompts, no closed-source layers. Audit, fork, modify.
```

### "Will this work in regulated industries (medical devices, pharma)?"

```
Not yet. We deliberately scope to consumer products (cosmetics, food, electronics, toys, textiles, supplements). Medical devices need MDR/IVDR-specific tooling and a different liability stance. On the roadmap for 2027.
```

### "What about Asia compliance?"

```
We cover JP (MHLW), KR (MFDS), CN (NMPA cosmetics) at Tier 5. India CDSCO and ASEAN (cosmetics directive) on the Q3 roadmap. Australia/NZ Q4.
```

### "Privacy / data handling?"

```
Zero PII collected by the skills themselves. The free API requires only an email. No telemetry, no analytics SDK, no phone-home in the skills. The MCP calls go to legaldata-public.cleolabs.co (logged for rate-limiting only, deleted after 30 days). Full privacy policy at cleolabs.co/privacy.
```

### "Roadmap?"

```
Public on the repo: github.com/Cleo-Labs-IA/skills_library/blob/main/ROADMAP.md. Highlights: Cursor port (Q3), Asia expansion (Q3-Q4), self-hosted enterprise mode (Q4), 50-skill milestone (Q1 2027).
```
