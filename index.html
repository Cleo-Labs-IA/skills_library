Les 12 skills

Tout ce dont tu as besoin pour vendre légalement

Chaque skill est un fichier Markdown autonome que Claude Code charge automatiquement. Pas de config, pas d'API key.

Tier 1 — Product Core
⚛ product-compliance HERO

Vérifie tes ingrédients contre 13 bases de substances (CosIng, ECHA, REACH, Prop 65, FDA, EFSA...). Verdict par substance par juridiction. Le cœur de la bibliothèque.

☷ labeling-compliance

Règles d'étiquetage pays par pays — INCI, allergènes, avertissements, personne responsable, CE/UKCA, multi-langue. Génère le label conforme.

✓ market-entry-checklist

Checklist étape par étape pour entrer sur un nouveau marché — de la classification produit à la première vente légale. Rien n'est oublié.

Tier 2 — Market Intelligence
⚠ regulatory-intelligence

Monitore les signaux qui peuvent bloquer ta vente — interdictions de substances, changements d'étiquetage, rappels, deadlines. Jamais pris au dépourvu.

☽ multi-jurisdiction-scan

Scan parallèle multi-agents sur tous tes marchés cibles. Un agent par juridiction, synthèse consolidée. 131 pays couverts.

✈ customs-and-trade

Classification HS, calcul de droits, coût rendu, screening dual-use, vérification sanctions. Tout pour franchir les frontières.

Tier 3 — Action
⚡ compliance-audit-sprint

Sprint compliance pré-lancement — identifie catégorie, mappe réglementations, vérifie substances, vérifie étiquettes, estime coûts. En 5 minutes.

⚒ compliance-remediation

Corrige les problèmes pour débloquer l'entrée sur le marché — reformuler, réétiqueter, tester, certifier. Passe du rouge au vert.

★ evidence-blitz

Collecte parallèle de toute la documentation compliance — docs sécurité, preuves étiquetage, docs substances, certifications. Tout en un run.

Tier 4 — Reference
☶ compliance-frameworks-ref

Référence rapide des réglementations produit — EU Cosmetics 1223/2009, REACH, FDA MoCRA, Prop 65, CE/UKCA, GPSR, EN 71, RoHS.

⚙ compliance-mcp-tools

Guide des 29 outils MCP — Cleo Legal API (customs, substances, sanctions) + Cleo Insight (signaux, réglementations) + Bastion.

▢ compliance-reporting

Matrice de conformité produit × marché. Exportable pour acheteurs, distributeurs, marketplaces. Le livrable final.

En action

Exemples concrets

Trois scénarios réels. Trois minutes chacun. Zéro expertise compliance requise.

claude-code — crème visage EU

Lancement d'une crème visage en Europe

Vérification de 23 ingrédients contre les annexes EU Cosmetics, génération du label INCI conforme, checklist d'entrée marché complète.

$ Vérifie ma formule pour vente en France : aqua, glycerin, niacinamide, retinol, phenoxyethanol, parfum, CI 77891... ■ Scanning 23 ingredients vs EU Cosmetics... ✓ 21/23 substances OK ⚠ retinol — max 0.3% (face) Annex III/217 ⚠ phenoxyethanol — max 1.0% Annex V/29 ✓ Label INCI généré (FR + EN) ✓ Personne responsable EU requise ✓ DIP (Dossier Information Produit) à préparer
claude-code — enceinte bluetooth US+EU

Certification d'une enceinte Bluetooth US+EU

Classification HS, marquage CE/FCC, test RoHS, screening dual-use, estimation des droits de douane.

$ Check compliance : enceinte BT portable Marchés : US + EU ■ Classification HS : 8518.22 (loudspeakers) ■ Scan multi-juridiction (2 agents)... EU : ✓ CE marking (RED 2014/53/EU) ✓ RoHS 2011/65/EU — conform ⚠ WEEE registration obligatoire US : ✓ FCC Part 15 (intentional radiator) ✓ Dual-use : not listed Droits : 4.9% (MFN)
claude-code — chocolat export Japon

Export de chocolat artisanal vers le Japon

Vérification des additifs alimentaires (MHLW), étiquetage en japonais, certificat sanitaire, droits de douane.

$ Export chocolat noir 72% au Japon Ingrédients : cacao, sucre, beurre cacao, lécithine soja, vanilline ■ Classification HS : 1806.32 ■ Scan Japon (Food Sanitation Act)... ✓ Tous additifs listés MHLW ⚠ Étiquetage JA obligatoire (7 allergènes) ⚠ Soja = allergène recommandé JP ✓ Certificat sanitaire FR requis Droits : 10% (WTO bound rate) ✓ Checklist 8 étapes générée

# Dans n'importe quel chat Claude Code
Vérifie la compliance de ma crème pour le marché EU
Scanne mes marchés cibles : FR, DE, US, JP
Génère un rapport compliance pour mon produit
