Le probleme

Le cauchemar compliance des petites boites produit

Mon ingredient est-il interdit en EU ?

13 bases de donnees, 500K substances, des limites differentes par pays. Aucun outil ne consolide tout ca pour une petite marque.

Quels labels pour vendre au Japon ?

Regles d'etiquetage differentes dans chaque marche. Allergenes, langues, pictogrammes, personne responsable -- tout change.

Combien ca coute d'entrer sur ce marche ?

Droits de douane, certifications, notifications, tests labo -- un labyrinthe administratif qui bloque les petites equipes.

La solution

12 skills qui repondent a une seule question

Qu'est-ce que je dois faire pour vendre mon produit sur ce marche ?

Product Core
product-compliance

Verifie tes ingredients contre 13 bases de substances (CosIng, ECHA, REACH, Prop 65, FDA, EFSA...). Verdict par substance par juridiction.

labeling-compliance

Regles d'etiquetage pays par pays -- INCI, allergenes, avertissements, personne responsable, CE/UKCA, multi-langue.

market-entry-checklist

Checklist etape par etape pour entrer sur un nouveau marche -- de la classification produit a la premiere vente legale.

Market Intel
regulatory-intelligence

Monitore les signaux qui peuvent bloquer ta vente -- interdictions de substances, changements d'etiquetage, rappels, deadlines.

multi-jurisdiction-scan

Scan parallele multi-agents sur tous tes marches cibles. Un agent par juridiction, synthese consolidee. 131 pays couverts.

customs-and-trade

Classification HS, calcul de droits, cout rendu, screening dual-use, verification sanctions. Tout pour franchir les frontieres.

Action
compliance-audit-sprint

Sprint compliance pre-lancement -- identifie categorie, mappe reglementations, verifie substances, verifie etiquettes, estime couts.

compliance-remediation

Corrige les problemes pour debloquer l'entree sur le marche -- reformuler, reetiqueter, tester, certifier.

evidence-blitz

Collecte parallele de toute la documentation compliance -- docs securite, preuves etiquetage, docs substances, certifications.

Reference
compliance-frameworks-ref

Reference rapide des reglementations produit -- EU Cosmetics 1223/2009, REACH, FDA MoCRA, Prop 65, CE/UKCA, GPSR, EN 71, RoHS.

compliance-mcp-tools

Guide des 29 outils MCP -- Cleo Legal API (customs, substances, sanctions) + Cleo Insight (signaux, reglementations) + Bastion.

compliance-reporting

Matrice de conformite produit x marche. Exportable pour acheteurs, distributeurs, marketplaces. Le livrable final.
