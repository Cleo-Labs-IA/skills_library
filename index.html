La stratégie

Comment on fait pour que tout le monde la télécharge

Pas juste un repo GitHub qui prend la poussière. Un plan d'acquisition en 6 canaux.

🔌
Canal 1 — Plugin Claude Code officiel
  • Soumission au registre de plugins Claude Code (Anthropic)
  • Si accepté = distribution à TOUS les utilisateurs Claude Code
  • Effet réseau immédiat sur toute la base installée
  • Action : PR vers le repo Anthropic plugins, respecter leur format
C'est le canal #1. Acceptation = distribution massive sans effort marketing.
🎯
Canal 2 — Claude.ai MCP Integration
  • Les skills utilisent Cleo Insight + Cleo Legal API via MCP
  • Chaque user qui installe les skills DÉCOUVRE nos MCP servers
  • Funnel : skills (gratuit) → MCP (gratuit) → Cleo Insight (payant)
Product-led growth pur. Le produit gratuit vend le produit payant.
🧵
Canal 3 — "Compliance in 30 seconds"
  • Série de posts courts (Twitter/LinkedIn/TikTok) : un prompt, un résultat
  • "J'ai demandé à Claude si ma crème au rétinol est légale en EU. Réponse en 8s."
  • Format : screen recording du terminal Claude Code → résultat compliance
  • 1 post/jour pendant 30 jours. Chaque post = 1 skill en action.
  • PAS de thread technique. Des résultats concrets pour fondateurs D2C.
🏪
Canal 4 — Shopify App Store + Marketplaces
  • Les skills sont le cerveau derrière l'app Shopify Cleo (Revenue-at-Risk)
  • Chaque marchant qui installe l'app utilise les skills sous le capot
  • Cross-pollination : "Powered by Comply skills" dans l'app
  • Extension : WooCommerce, BigCommerce, Prestashop
🎓
Canal 5 — Compliance Academy
  • "Lancer sa marque cosmétique en EU : le guide complet"
  • "Exporter au Japon : de A à Z"
  • "CE marking pour les nuls"
  • Chaque guide se termine par "Automatisez tout ça avec skills_library"
  • Blog Cleo, Medium, Notion templates, newsletters D2C
  • SEO long-tail sur les requêtes compliance produit
Éducation comme canal d'acquisition. Le contenu amène le trafic, les skills convertissent.
🤝
Canal 6 — Partenariats verticaux
  • Labs de test (SGS, TÜV, Intertek, Eurofins) : "Recommandez nos skills à vos clients avant les tests"
  • Incubateurs D2C (Station F, Y Combinator, Techstars) : "Le toolkit compliance de vos startups"
  • Plateformes d'ingrédients (Aroma-Zone, MakingCosmetics) : "Vérifiez la conformité avant de formuler"
  • Réseaux de Responsible Persons : "Pré-qualifiez vos clients avec nos skills"
Chaque partenaire = un canal de distribution avec crédibilité intégrée.

Exemples concrets

claude-code
$Vérifie ma crème visage rétinol pour le marché EU
  • ✓ 21/23 substances OK (CosIng + REACH)
  • ! Rétinol — max 0.3% (Annex III/217)
  • ! Phenoxyethanol — max 1.0% (Annex V/29)
  • ✓ Notification CPNP requise
  • ✓ Responsible Person EU à désigner
  • ✓ DIP (Dossier Information Produit) à préparer
claude-code
$Enceinte Bluetooth — certifications US + EU
  • ✓ CE marking (RED 2014/53/EU)
  • ✓ FCC Part 15 Subpart C
  • ✓ RoHS 2011/65/EU — conforme
  • ! WEEE registration obligatoire (chaque État membre)
  • ✓ HS : 8518.22 (loudspeakers)
  • ✓ Droits : 4.9% (MFN EU), 0% (US)
claude-code
$Complément alimentaire export Japon
  • ✓ Food Sanitation Act — catégorie "health food"
  • ! Étiquetage JA obligatoire (7 allergènes majeurs)
  • ! Déclaration d'importation MHLW requise
  • ✓ HS : 2106.90 (préparations alimentaires)
  • ✓ Certificat sanitaire pays d'origine requis
  • ✓ Droits : 8.8% (WTO bound rate)
